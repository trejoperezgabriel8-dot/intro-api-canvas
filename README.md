# 🎨 Proyecto: Rectángulo Estilizado con Canvas

**Autor:** Gabriel Trejo Pérez  
**Fecha:** 16/02/26  
**Materia:** Graficación / Programación Web  

---

## 📌 Descripción del Proyecto

Este proyecto demuestra el uso del elemento `<canvas>` de HTML5 junto con JavaScript para dibujar un rectángulo personalizado.

El programa incluye:

- Ajuste dinámico del canvas al 50% del ancho y alto de la ventana.
- Rectángulo con:
  - Esquinas redondeadas
  - Sombra
  - Borde color café oscuro
  - Relleno color café claro
- Estilización visual mediante CSS externo.

---

## 🛠 Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript
- API Canvas 2D

---

## 📂 Estructura del Proyecto

📁 Proyecto-Canvas
│── index.html
│── script.js
│── estilos.css
│── README.md


---

## ⚙ Funcionamiento

1. El archivo HTML contiene el elemento `<canvas>`.
2. El archivo CSS aplica estilos visuales al canvas.
3. El archivo JavaScript:
   - Ajusta el tamaño del canvas al 50% de la ventana.
   - Configura colores y sombra.
   - Dibuja un rectángulo redondeado utilizando métodos de la API Canvas.

---

## 🎨 Características Visuales

- Fondo del canvas gris claro.
- Rectángulo café claro.
- Borde café oscuro.
- Sombra proyectada.
- Esquinas redondeadas mediante curvas cuadráticas.

---

## 📐 Sistema de Coordenadas

El canvas utiliza un sistema de coordenadas donde:

- (0,0) está en la esquina superior izquierda.
- El eje X crece hacia la derecha.
- El eje Y crece hacia abajo.

---

## 🚀 Cómo Ejecutar el Proyecto

1. Descargar los archivos.
2. Abrir `index.html` en el navegador.
3. El canvas se ajustará automáticamente al tamaño de la ventana.

El propósito de este proyecto es comprender:

## 📌 Conclusión
